Geant4 Server on Docker Hub

Remora's Geant4 server represents an example Geant4 application that can connect to the Remora client. To use, simply run the Geant4 app in a docker container on the port 8080. Then open one or several Remora clients and watch as the geometries and particle trajectories of the Geant4 app are rendered.
